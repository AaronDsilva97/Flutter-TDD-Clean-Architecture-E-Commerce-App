## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Sameera-Perera/EShop-Flutter-Clean-Architecture.git
   ```
2. Install packages
   ```sh
   flutter pub get
   ```
